# LED Chessboard

## Client

### Technologies

- React (Vite)
- Typescript
- TailwindCSS
- Chess.JS

### In the `/client` directory:

Install the dependencies

```
npm i
```

Run the React Vite app

```
npm run dev
```

## Server

### Technologies

- Node.JS
- Javascript
- Socket.IO
- Stockfish
- Chess.JS
- Firebase

### In the `/server` directory:

Install the dependencies

```
npm i
```

Run the Node.JS server

```
nodemon
```
